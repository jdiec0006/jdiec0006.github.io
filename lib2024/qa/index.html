  1. Qus 1: What are the three componets of Web?

  2. Qus 2: Define version control and give one example.

    Ans: Version control is a system that records changes on files and allows you to access old versions of the same files as well. IE//GitHub

  3. Qus 3: What tag is used to create the title of a web page? Give an example.

    Ans: < title > tag is used to create a title.

  4. Qus 4: What tag is used to create a paragraph in html? Give an example.

    Ans: < p > tag is used to create paragraphs.

  5. Qus 5: Identify the odd one from these tags: < span >, < p >, < a >, < i > and state the reason?

    Ans: < a > defines a hyperlink while the other tags has to do with text and styling.

  6. Qus 6: What are the tags required to create a navigation list in HTML?

    Ans: < nav >

  7. Qus 7: What element is used to create images? Name its attributes.

    Ans: < img > is the element used for creating images.

  8. Qus 8: Identify the odd one from these tags: < table >, < tr >, < b >, < td > and state the reason

    Ans: < b > is a tag for making a break in HTML while the other tags putains to creating a table.

  9. Qus 9: Give an example of color picker input.

    Ans: < input type="color" > is an example of a colour picker input.

  10. Qus 10: What is the use of name and value attributes? Give two examples.

    Ans: Name attribute is used for identifying the element such as feildset or text area. Value attribute specifies the value of the element such as input or option.
    ie// < input type="text" name="username" value="john" / >

  11. Qus 11: Give an example of inline style.

    Ans: Inline style is applied to a single html element such as h1.

  12. Qus 12: How would you link an external style sheet? Give example.

    Ans: You would link an external style sheet by adding a link in the < head >.
    < link rel="stylesheet" href="styles.css" >

  13. Qus 13: Create CSS block to center an element both horizontally and vertically that has position:absolute and has width:200px and height:200px.

    Ans: .element-name { position: absolute; left: 50%; top: 50%; width: 200px; height: 200px; }

  14. Qus 14: Create CSS keyframes block called "spin" that rotates an element.

    Ans: .loader { animation: spin 0.25s linear infinite; }

  15. Qus 15: Create CSS animation block to rotate an element that has the following properties :

    a) class "rotate-me"

    b) It rotates 5 times at intervals of 0.5s with "linear" transition using animation keyframes "spin"

    Ans: .rotate-me { animation: spin 0.5s linear 5; } @keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } }

  16. Qus 16: How would you include create an alert with string "I love JavaScript" using JavaScript?

    Ans: < script > function myFunction() { alert("I love JavaScript"); }
