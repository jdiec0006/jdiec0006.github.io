  2. Qus 2: Define version control and give one example.

    Ans: Version control is a system that records changes on files and allows you to access old versions of the same files as well. IE//GitHub

  3. Qus 3: What tag is used to create the title of a web page? Give an example.

    Ans: < title > tag is used to create a title.

    Ans: < p > tag is used to create paragraphs.

  6. Qus 6: What are the tags required to create a navigation list in HTML?

  7. Qus 7: What element is used to create images? Name its attributes.

  9. Qus 9: Give an example of color picker input.

    Ans: Name attribute is used for identifying the element such as feildset or text area. Value attribute specifies the value of the element such as input or option.
    ie// < input type="text" name="username" value="john" / >

  11. Qus 11: Give an example of inline style.

    Ans: Inline style is applied to a single html element such as h1.

  12. Qus 12: How would you link an external style sheet? Give example.

    Ans: You would link an external style sheet by adding a link in the < head >.
